Arduino 练习备忘录
==================

硬件信息
--------

[Arduino Uno R3][1]

![Arduino Uno R3 Front][2]

![Arduino Uno R3 Back][3]



练习规则
--------

 - 每天次。即，每次学习只要是当天内的，都算做一次。
 - 非线性。即，每天次并不一定是连续的自然日。


练习目标
--------

 - 理解 Arduino 基础理念。
 - 掌握 Arduino 核心方程。
 - 精通 Arduino 基本库使用。
 - 了解 Arduino 其他第三方库，但不限于以下库。
    - 通讯。
    - 传感。
    - 显示、LED。
    - 音频及波形。
    - 定时器。


归档形式
--------

 - 前缀 `dxxxx`: `d` 代表天。 `xxxx`代表次数。
 - _主题内容，多个使用关键字_
