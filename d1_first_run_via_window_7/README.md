初试牛犊
========

Windows 7 下安装及测试
------------------------

1. 在[官方下载页][1]，下载最新发布版本，当前撰写日为2013/8/25，版本 1.0.5。
2. 最省心的方式即可下载 [Windows Installer][2]，适合全新的安装。
3. 安装完成后即可通过 USB 连接线，和 PC 连接上 Arduino 进行测试。
4. 启动 Arduino 应用程序。
5. 界面支持中文，设置方法： `File` > `Preferences` > `Editor language` > `简体中文（Chinese Simplified）`.
6. 重启 Arduino 。
7. 选择 `文件` > `示例` > `01.Basics ` > `Blink ` 后，可以看到示例代码。
8. 确认 `工具` > `板卡`  所选择的是  `Arduino Uno` 。
9. 确认 `工具` > `串口`  所选择的类型和 windows 设备管理器下显示的 `Arduino Uno端口（COMxxx）`  中的 COM 类型一致，如 COM3。
10.点击下载（确切翻译为“上传”）。此时板卡上 RX 和 TX 会开始闪烁。之后，编辑界面显示“下载完毕”。即可看到 13 针脚（L）的 橘色 LED 灯闪烁。
11.变更代码中  delay(1000);中的数值，即可看到按毫秒（1000毫秒 = 1秒）作用于前一句的延迟响应。


基础知识概要（1）
----------------

### Sketch / 速写

 示例：在《Windows 7 下安装及测试》中，第 7 步中的 Blink 。

- Arduino 程序的称呼。
- 一个 Arduino 板卡的上传代码单元。
- 由注释、变量、函数组成。
    - 注释。由 `/* 多行注释 */` 和 `\\单行注释` 组成。
    - 变量。存在全局变量和局部变量，通过类型申明。`int led = 13;`。
    - 函数。`setup()` 和 `loop()`两个特殊函数是每份 Sketch 的必需确保存在的组成部分。
        - `setup()` 程序的主入口。仅运行一次，一般用于初始化程序。
        - `loop()` 程序的心跳函数。重复运行，猜测运行周期等于芯片执行周期。

### 微控制器

#### 针脚

- Arduino 所有数字针脚都可以配置用于输入或者输出。
- Arduino 大多数模拟针脚也可以像数字针脚一样使用。

#### 数字针脚

- 配置为输入
    - 默认配置是作为输入使用，因此不需要用 `pinMode()` 特别指定。
    - 意味着高阻抗。
    - 上拉电阻能够提供足够的电流让输入针脚朦胧地点亮 LED，这往往意味着忘记将针脚通过 `pinMode()`设置为输出 。
- 配置为输出
    - 意味着低阻态。
    - 外接设备时建议连接 470Ω 或 1K 电阻确保不会因短路或电流过大而导致 Arduino 芯片击穿。
- 13号数字针脚
    - 相对其他数字针脚作为输入针脚是非常艰难的。因为在大多数板卡上都焊接这一个 LED 和电阻。
    - 如果非要用 13 号数字针脚作为数字输入针脚，则需要用额外的下拉电阻。
